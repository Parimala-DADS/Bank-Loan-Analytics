# Bank Loan Analytics Dashboard
Bank loan analysis using Power BI to track loan performance and customer insights

## Project Overview
This project analyzes bank loan data to understand loan performance, customer behavior, and financial risk.
An interactive Power BI dashboard was developed to visualize key loan metrics and trends.

## Tools & Technologies
- Power BI
- Excel
- SQL
- Data Visualization
- Data Analysis

## Key Metrics
- Total Loan Applications: 38.3K
- Total Funded Amount: $430.6M
- Total Interest Paid: $4.6K
- Average Interest Rate: 12%
- Total Monthly Payment: $39.98M

## Dashboard Pages
### 1. Summary Page
- Good vs Bad loan analysis
- Loan status and recovery efficiency
- Loan distribution by religion
- Interest and exposure analysis

### 2. Overview Page
- Monthly loan amount trends
- State-wise loan distribution (Map)
- Loan analysis by term, age group, purpose, and home ownership

- 74.5% of loans are classified as good loans
